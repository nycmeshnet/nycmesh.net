# NYC Mesh Website

## Install

Install [Hugo](https://gohugo.io):

```bash
brew install hugo
```

## Usage

Start the local server:

```bash
hugo serve
```

Navigate to the site at http://localhost:1313.

## Adding Content

Content is stored as markdown in `content`. 

New blog:

```bash
hugo new blog/post-title.md
```

New press:

```bash
hugo new press/article-title.md
```
